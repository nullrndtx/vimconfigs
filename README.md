### Installation

```sh
* clone this repository - git clone https://github.com/nullrndtx/vimconfigs ~/.vim
* symlink vimrc from ~/.vim to ur home - ln -s ~/.vim/vimrc ~/.vimrc
* start using vim, type vim in ur terminal and wait until finish
```
